# Caro
A game like Tic Tac Toe but you need 5 in a row to win.

## Project
Made with Express.js, JsonDB, EJS and Socket.io.
For you guys that don't know what is JsonDB, it's a part of my framework. See [Newer.js](/posts/Newerjs) for more info.

## Features
- Multiplayer
- Rematch
- Spectate
- Log in
- Scoreboard
